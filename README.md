# 🧾 나만의 명대사 영수증 생성기 (Receipt Maker): Booty Wurk & Full Bloom ʚ🤍ɞ

> **"부티워크와 풀 블룸의 명대사를 영수증 한 장에 랜덤으로 뽑아보세요!"** 🎲✨
> ꒰☁️꒱ 작품 속 잊지 못할 명문장을 영수증 디자인으로 만나보고, 이미지로 소장할 수 있는 사이트입니다.

---

## ✨ 알아두면 좋은 디테일 포인트!

* 🎲 **오늘의 한 문장 랜덤 뽑기**: 부티워크와 풀 블룸, 둘 중 어디의 명대사가 나올지는 완전히 무작위(Random)! 새로 뽑기를 할 때마다 운명처럼 다가오는 문장들을 만나보세요.
* 🌡️ **작품별 온도 디테일**: 
  * 🧊 **부티워크** 대사가 나올 때는 온도가 **18도**로 표시됩니다.
  * 🌸 **풀 블룸** 대사가 나올 때는 온도가 **22도**로 표시됩니다.
  * 온도에 맞춰 바코드 위에 적힌 문장들도 다르게 출력되는 소소한 디테일을 찾아보세요! 🔍
  * 무슨 온도인지, 왜 온도가 변하는지는 구부단이라면 다들 아시리라 생각합니다 😎
* 📸 **영수증 뽑기**: 마음에 드는 영수증이 나왔다면 우측 상단의 **이미지 저장📥** 버튼을 눌러 갤러리에 쏙 저장해 보세요.

## ⚠️ 이용 안내 및 오류 해결 방법 (FAQ)

사이트를 이용하시면서 궁금하실 수 있는 점들을 정리해 두었어요! 🤓

### 1. 🚦 혹시 사이트가 터지진 않을까요?
명대사를 불러오기 위해 구글 시스템을 사용하고 있어서 하루 이용 횟수 제한이 살짝 존재합니다(하루 스크립트 실행 횟수 제한 2만번). 하지만 그 제한에 걸릴 만큼 엄청나게 많은 분들이 동시에 몰릴 일은 아마 없을 테니...(ㅠㅠ) 걱정 말고 마음껏 즐겨주세요! 😉 (만에하나 제한에 걸리더라도, 다음 날(미국 태평양 표준시(PST) 자정을 기준으로 리셋=한국 시간(KST)으로는 오후 4~5시경)이 되면 정상적으로 작동합니다!)

### 2. 🛡️ 보안은 안전한가요? (바이러스/해킹 걱정 NO!)
걱정 안 하셨을 수도 있지만 혹시나 걱정하실 분들이 계실까봐...
완벽하게 안전하니 안심하고 사용하세요! 🙅‍♀️ 사이트 내에 바이러스나 악성 코드는 전혀 없습니다. 또한, 누군가 악의적으로 사이트 해킹을 시도하더라도(그럴 이유가 있을지는 모르겠지만...) 시스템 상 절대 작동하지 않습니다.

예를 들어 박정대라는 해커가 어떻게든 구글 시트에 접근해서, 명대사 대신 사람들의 기기를 망가뜨릴 이상한 해킹 코드 같은 것을 억지로 적어넣는다 한들(굳이?) 사이트 내에서 그 코드가 실행되진 않습니다. 그냥 코드 영문 자체가 영수증 안에 인쇄되어 나올 뿐...

ex. 오늘의 문장: <script> 바이러스 실행 </script>

박정대라는 이름의 해커가 할 수 있는 최대치의 나쁜 짓은 기껏해야 시트 안에 적힌 명대사 데이터를 통째로 긁어가는 것뿐입니다 🥹 

**박정대**: 뭐야?

### 3. 📱 이미지 저장 버튼이 안 눌리거나 영수증이 잘려요! (모바일)
트위터(X), 인스타그램, 카카오톡 등 **앱 내장 브라우저**로 접속하셨을 경우, 해당 앱들의 자체 환경 때문에 다운로드가 막히거나 이미지가 살짝 어긋나게 캡처될 수 있습니다. 
👉 **해결법**: 화면 우측 상단이나 하단의 메뉴 버튼(`...`)을 눌러 **'다른 브라우저로 열기(Safari, Chrome, 삼성 인터넷 등 기본 브라우저)'**를 선택해 접속하시면 아주 깔끔하게 저장됩니다! ✨

### 4. 영수증 폰트가 예시 사진이랑 다르게 나와서 글자가 밀려요! (모바일)

사용중인 브라우저의 폰트 설정이 기본 폰트가 아닐 경우 제목 폰트가 밀릴 수 있습니다. 큰 문제는 아니지만 예쁜 영수증을 받아보시고 싶으시다면 기본 폰트로 설정해주세요 ദ്ദിᐢ. .ᐢ₎
